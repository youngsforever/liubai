# 六个 macOS 装机必备的应用

嫌弃我的 MacBook Air 太卡，想重置一下操作系统；在重新捣鼓配置时，顺手记录一下我觉得 2026 这个时间节点，有哪些经过时间检验必须安装的软件。

以下 6 个皆为个人心得，仅供参考。

## 1. Shottr

[https://shottr.cc/](https://shottr.cc/)

> 快速截图工具

我常用它的 `Cmd + Shift + 2` 快速截图。这几个快捷键按下去即可选择截图区域，划完后图片将存在剪贴板中，我再点开 [留白](http://my.liubai.cc/) 的窗口按下 `Cmd + V` 直接贴上。

快速捕捉屏幕内容，真的超方便！

## 2. Input Source Pro

[https://inputsource.pro/zh-CN](https://inputsource.pro/zh-CN)

> 输入法切换提示

因为我会用注音、拼音和英语输入，需要在选中输入框时快速识别当下是哪个输入法，用 Input Source Pro 弹出输入法气泡即可得知。

## 3. Mos

[https://mos.caldis.me/](https://mos.caldis.me/)

> 让鼠标滚动更加顺滑

用鼠标上的滚轮 🖱️ 滚动时，如果没有 Mos 用起来会非常卡顿，每滚一格就卡住，但有了它就自然许多。

## 4. Hidden Bar

[https://apps.apple.com/cn/app/hidden-bar/id1452453066](https://apps.apple.com/cn/app/hidden-bar/id1452453066)

> 隐藏无用的菜单按钮

不少应用都爱占用菜单栏空间，导致视觉凌乱。Hidden Bar 能帮你隐藏不常用的图标，让顶部状态栏保持清爽，同时让核心工具更易于寻找。

## 5. Raycast

[https://raycast.com/](https://raycast.com/)

> 快速访问工具

我都用它的 `Option + Space` 快速唤起搜索框，然后输入 `emoji` 去选择我想要的表情。

它的 `emoji` 是支持语义搜索的，输入 `smile` 就会找出笑脸相关的表情，输入 `cat` 就会找到猫猫。很实用！

## 6. LocalSend

[https://localsend.org/zh-CN](https://localsend.org/zh-CN)

> 开源的局域网文件传输工具

直接用 AirDrop 传输图片时常得到的是 .heic 格式，用 LocalSend 传输相册里的图片，会得到 .jpg / .png 格式。
