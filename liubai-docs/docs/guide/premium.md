# 留白记事 Premium

以下是 `留白记事 Premium` 所有权益：

## [Web App](https://my.liubai.cc)

|                 |      **免费版**      |      **Premium**      |
| --------------- | ------------------- | --------------------- |
|     笔记数       |  50 <sup>注1</sup>  |         无限制         |
|     置顶数       |          1          |          3            |
|  单个文件容量     |         10 MB        |        100 MB        |
|  单个动态里图片数  |          2           |         9            |

`注1` 即使数量达到上限，也不会阻止使用，而是自动删除最旧的笔记；尚未实施。


## [小程序](/article/2025/weixin-task)

|                                |      **免费版**        |      **Premium**      |
| ------------------------------ | --------------------- | --------------------- |
|     任务数 <sup>注1</sup>        |          3           |         无限制       |
|   手动选择时间 <sup>注2</sup>     |          无           |           有          |
|   添加到手机日历 <sup>注3</sup>   |          无           |           有          |

`注1` 仅统计 **有效任务**（未完成、未结束），不含 **归档的任务**（已完成、已结束）。

`注2` 修改标题 AI 就会自动识别时间；时间选择器仅对 Premium 开放。

`注3` 将任务添加到手机日历后，若任务发生变化，无法实时同步到你的日历上。故该功能只面向有使用手机日历的朋友，有需要再购买。
